Git

廖雪峰
1. 初始化Git仓库：git init
2. 添加文件到Git仓库：
1) git add <file> 可以反复多次使用，添加多个文件，rm删除，然后也需要提交
2) git commit -m <message>
3. 监管仓库
1) git status 能让我们时刻掌握仓库的当前状态，可以告诉我们问价被修改过
2) git diff 可以查看修改的内容
4. 时空穿梭
1) HEAD （指针）指向的是当前版本，git reset --hard commit_id 修改版本
2) 修改之前，用git log 可以查看提交历史，以便要退到哪个版本
3) 要重返未来，用git reflog 查看命令历史，以便确定回到未来的哪个版本
5. 设置本地信息
git config --global user.name ‘your_name’
git config --global user.email ‘your_email’
