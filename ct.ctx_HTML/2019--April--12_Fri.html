javaWeb


1. JavaWeb
1) response.setContentType("text/html; charset=utf-8);
2) request.setCharacterEncoding("utf-8);
3) initParams={@WebInitParam(name, value)} + getInitParameter(name);
4) 通过request的一系列getXXX()函数，可以获得服务器信息、servlet头信息等等
1- request.getHeaderNames(); request.getHeader(name);
2. Form表单
1) input<type ="text" maxlength="8" size="16">maxlength设置字符最大长度
2) radio名字一样，直接写checked表示默认选项
3) select可以多选可以双选(+multiple size=""，使用ctrl+左键多选)
4) Submit按钮
1- 如果加了name属性，则在提交的时候会把其name 和value属性作为表单数据一同提交上去
2- 如果多个按钮name相同，value不同，在提交表单的时候可以根据value值进行分支处理。
3. css id选择器和class选择器
1) id="para1" -> #para1{...};使用id也可以进行跳转<a href="#para1">
2) class="asd" p.ass{...} / .asd{...}
4. 获得表单数据的函数:request.getParameter(name);
5. 自定义！filterHtml()；对提交的参数值进行过滤，过滤到脚本语言; '\n' → ‘<br>’
6. document.getElementById();用来获得指定ID的第一个对象
