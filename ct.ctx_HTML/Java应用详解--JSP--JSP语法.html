JSP语法

1. 注释
1) 显式注释（HTML风格）<!-- ... -->
2) 隐式注释
1- //...
2- /* ... */
3- <%-- ... --%>
2. scriptlet标识（所有嵌入到HTML的Java程序都必须使用scriptlet标识出来）
1) 代码片段（Java代码或者脚本代码）: <% int i=10; %>;
2) 声明标识（JSP页面全局的变量或者方法）:<%! %>; 可以不在一行
3) JSP表达式（向网页输出信息）:<%= incrementCounter() %>; 函数结束后没有';'
3. 指令标识
1) page指令：可以定义一个JSP页面的相关属性（只有import指令可以出现多次，其他的只能一次）
1- 设置页面的MIME: <%@ page contentType="text/html; charset=utf-8"%>
2- 设置文件编码: <%@ page pageEncoding="utf-8"%>
3- 错误页的设置：<%@ page isErrorPage="true"%> <%@ page errorPage="/error.jsp"%>
4- 数据库连接操作：<%@ page import="java.io.*, java.util.*" %>
2) include指令：可以在一个JSP页面包含另一个JSP页面（静态包含，原样包含，所以不能有同名变量）
1> 只有一个属性file:<%@ include file="header.html"%>
4. 动作标识
1) <jsp:include>包含动作：向当前页面包含其他文件，可以是动态，可以是静态
1- <jsp:include page="url" flush="false|true"/> flush是可选属性
2) <jsp:forward>请求转发动作: 指定请求转发的页面，必须是当前应用中的资源
1- <jsp:forward page="url"/>
5. 内置对象（无需声明就可以直接使用的对象）

1) session：客户端与服务端的状态，在用户离开Web应用之前一直有效
2) Application：整个Web应用运行期间的全局数据和信息
