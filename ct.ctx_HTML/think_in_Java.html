think in Java

1. Java 与c++ 相比是纯粹的面对对象的语言
2. Java 默认的访问权限是：同包可以互访，包外就是private
3. 存储位置：
1) 寄存器，最快但是内存小
2) 堆栈：位于通用RAM， 限制了灵活性，但是快，c++
3) 堆：通用的内存池，也在RAM,灵活，但是慢 java
4) 常量存储：存于程序代码内部
5) 非RAM存储：完全独立于程序
4. Java 中 char占 2 bytes
5. boolean 无固定大小，只能取字面值，true or false
6. 没有无符号整型
7. BigInteger and BigDecimal
1) b1.add(b2);
2) b1.subtract(b2);
3) b1.multiply(b2);
4) b1.divide(b2);
5) b1.divideAndRemainder(b2);返回值是BigInteger[],结果和余数
9. 不能在程序块中，隐藏外部的变量 int a;{int a;}
10. Java对象，可以存活在作用域之外
11. Java数组会自动初始化，清零
12. Java 实现多组输出
6) while(sc.hasNextInt()) or while(sc.hasNext())
13. 计算时间，单位是毫秒？
7) long startTime = System.nanoTime();
14. Java 判断数字和字符串和转换：
8) String.valueOf(int) int -> String
9) Integer.valueOf(int or String) str -> int
10) 如果失败会抛出 NumberFormatException 异常
15. Java sort 排序
11) import java.util.Arrays;
12) Arrays.sort(arr, 0, n, cmp());
16. Java 只有类定义的变量才会自动进行初始化，在函数内的局部变量不会 和 c++一样
